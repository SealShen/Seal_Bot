## MCP 設定稽核步驟

（稽核觸發條件由 CLAUDE.md 全域規則管理，此處僅定義執行方式）

1. 確認 MCP 設定位置：`~/.claude.json` 的 mcpServers 區塊（**僅讀取非憑證欄位**）
2. 逐項檢查：
   - 工具範圍是否符合最小權限原則
   - env 憑證是否僅存於 `~/.claude.json`，未硬編碼於任何腳本或程式碼
   - Server 是否為本機執行（禁止連接外部 MCP endpoint）
3. 對照 ISMS 規範（依需要定向讀取相關章節）
4. 產出稽核紀錄：`output/security/mcp_audit_{YYYYMMDD}.md`

### 憑證管理硬性限制

- **禁止**讀取 `.mcp.json`
- **禁止**讀取、列印 `.env`、`.pem`、`.key`、`.p12` 等憑證檔案內容
- **禁止**讀取 `~/.claude.json` 中 env 的憑證值（KEY、TOKEN、SECRET、PASSWORD）
- MCP 設定異動後，由使用者手動確認憑證存放正確，Claude 不做驗證
